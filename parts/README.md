Here are the 12 core K'Nex part JSON files — ready to drop straight into your parts/ folder.
They follow exactly the schema we defined earlier (schema/knex-part.schema.json).

Dimensions are real-world accurate (mm) from knexusergroup.org.uk + Instructables measurements.
Connector arm radius = 12.5 mm from center to port (standard K'Nex geometry).
Rod lengths = actual measured body length; when snapped, center-to-center distance = rod_length + 25 mm (2 × 12.5 mm).
Colors = classic K'Nex factory colors.
Masses = approximate real-part weights (grams).
mesh_file paths assume you’ll create the GLBs next (I can give you the exact OpenSCAD scripts for each if you want).
